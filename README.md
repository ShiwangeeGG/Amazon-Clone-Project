# AMAZON WEBPAGE CLONE PROJECT 🛒

## Overview
This project is a front-end user interface clone of the Amazon e-commerce homepage. It was built to practice and solidify foundational web development skills, specifically focusing on page structure, layout techniques, and styling a complex, real-world user interface.

## Technologies Used
* **HTML5:** Used for structuring the web page and organizing content semantically.
* **CSS3:** Used for styling, positioning elements (like Flexbox), and matching the official visual design.

## Acknowledgements
This project was built by following the excellent HTML & CSS Amazon Clone tutorial by **Apna College** on YouTube. 

## Features
* Replicated Amazon's top navigation bar, including the search input and logo placement.
* Main hero section with background banner.
* Product grid layout featuring individual item cards.
* Replicated, multi-column footer section.

## How to View
To view this project locally on your machine:
1. Clone this repository: `git clone https://github.com/ShiwangeeGG/Amazon-Clone-Project.git`
2. Open the project folder.
3. Double-click the `index.html` file to launch it in any modern web browser.
